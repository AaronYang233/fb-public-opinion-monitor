# 情感分析规则

## 情感等级

| 等级 | 标签 | 说明 |
|------|------|------|
| 1 | 强负面 | 情绪激动、使用强烈负面词汇、涉及投诉/曝光 |
| 2 | 负面 | 表达不满、批评、问题描述 |
| 3 | 中性 | 客观描述、无明显情绪倾向 |
| 4 | 正面 | 满意、表扬、推荐 |

## 负面关键词库

### 强负面词汇（触发 "强负面" 判断）

```
退货、退款、投诉、举报、欺诈、骗子、假货、劣质、
质量差、做工差、货不对板、严重、致命、完全失败、
垃圾、废物、毫无用处、欺骗消费者、三无产品、
有毒、危险、安全隐患、爆炸、起火、过热
```

### 负面词汇（触发 "负面" 判断）

```
不满、失望、差、糟糕、难用、不好、问题、故障、
损坏、有瑕疵、划痕、变形、褪色、异味、不舒服、
延迟、很久没到、太慢了、客服态度差、推诿、
不处理、没人管、希望改进、需要加强
```

### 正面词汇（触发 "正面" 判断）

```
满意、很好、优秀、超值、推荐、喜欢、实用、物美价廉、
性价比高、客服好、服务周到、解决问题、响应快、
好用、方便、漂亮、好看、正品
```

## 问题分类规则

### 分类定义

| 分类 | 关键词/场景 |
|------|------------|
| **质量问题** | 破损、瑕疵、异味、褪色、变形、材质差、做工粗糙、假货 |
| **功能异常** | 不工作、故障、不能用、功能缺失、兼容性问题、充不进电 |
| **售后问题** | 退货难、退款慢、客服不理、推诿、拒绝保修、维修收费高 |
| **安全风险** | 漏电、起火、爆炸、有毒、过热、夹手、锐利边缘 |
| **其他** | 物流问题、价格虚高、虚假宣传、包装破损 |

### 分类优先级

当一条帖子包含多个分类关键词时，按优先级取第一个匹配：
1. 安全风险（最高优先级）
2. 质量问题
3. 功能异常
4. 售后问题
5. 其他

## 严重程度评估

### 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 情感强度 | 30% | 强负面=4, 负面=3, 中性=2, 正面=1 |
| 传播潜力 | 30% | 点赞+评论数 ≥100=4, ≥50=3, ≥10=2, <10=1 |
| 问题类型 | 25% | 安全=4, 质量=3, 功能=3, 售后=2, 其他=1 |
| 用户影响力 | 15% | KOL/粉丝多=4, 普通=2, 小号=1 |

### 严重程度等级

| 综合评分 | 严重程度 | 说明 |
|---------|---------|------|
| 3.5-4.0 | 严重 | 需立即处理，可能上热搜 |
| 2.5-3.4 | 高 | 需24小时内响应 |
| 1.5-2.4 | 中 | 需3天内处理 |
| 1.0-1.4 | 低 | 常规监控 |

## 扩散风险评估

| 条件 | 扩散风险 |
|------|---------|
| 点赞≥100 OR 评论≥50 | 高 |
| 点赞≥50 OR 评论≥20 | 中 |
| 其他 | 低 |

## 根因推测规则

根据关键词组合推断根因：

| 问题模式 | 推测根因 |
|---------|---------|
| 多个帖子提及"异味"+"味道怪" | 原材料问题/存储问题 |
| 多个帖子提及"充不进电"+"不能用" | 产品质量缺陷/批次问题 |
| 多帖子提及"客服不理"+"退货难" | 售后服务流程问题 |
| 多帖子提及"爆炸"+"起火"+"过热" | 严重安全隐患，紧急处理 |
| 多帖子提及"假货"+"货不对板" | 渠道/供应链问题 |

## 建议生成规则

| 分类 | 建议模板 |
|------|---------|
| 质量问题 | "建议联系售后换货，保留购买凭证，如多次投诉建议抽检同批次产品" |
| 功能异常 | "建议提交工单技术支持，必要时申请退换货，排查是否为个例" |
| 售后问题 | "建议升级投诉级别，联系总部客服，记录工单编号跟进" |
| 安全风险 | "⚠️ 立即下架相关批次产品，进行安全检测，发布官方声明" |
| 其他 | "建议收集更多案例，分析是否为个案或系统性问题" |
